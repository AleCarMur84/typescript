# Backend API - Documentación

## Axios
Axios es una librería para hacer peticiones HTTP desde el cliente o Node.js. Facilita el manejo de promesas y respuestas JSON.

## Postman
Postman es una herramienta para probar APIs sin necesidad de frontend. Permite enviar requests HTTP y ver respuestas.

## Sentry
Sentry es una herramienta de monitoreo de errores en producción. Permite detectar fallos en tiempo real.

## Swagger
Swagger es una herramienta para documentar APIs REST de forma visual e interactiva.

## Uso en proyectos
Estas herramientas se usan para:
probar APIs
documentar endpoints
detectar errores
mejorar la comunicación entre frontend y backend
